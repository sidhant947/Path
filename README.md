
# Path

Path is a lightweight, open-source step tracking application designed with a focus on privacy and simplicity. Track your daily activity without unnecessary bloat or data collection.

## Features

- **Privacy First**: All data is stored locally on your device.
- **Battery Efficient**: Uses hardware sensors to minimize power consumption.
- **Clean UI**: Simple visualizations of your daily and weekly progress.
- **Open Source**: Transparent codebase and community-driven.
